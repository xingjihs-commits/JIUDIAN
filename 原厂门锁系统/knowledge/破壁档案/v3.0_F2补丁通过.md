## F2 补丁技术细节

**地址映射**：

```text
VA 0x0068218F  ←→  file offset 0x28158F
原字节: 3B F0 0F 85 9E 02 00 00     ; cmp esi,eax ; jne 0x682435 (失败跳转)
补丁后: 3B C0 0F 85 9E 02 00 00     ; cmp eax,eax ; jne 永远不跳，进 0x682197 工程分支
```

**周围逻辑**（来自 capstone 反汇编 `CardLock.exe`）：

```text
0x00682105  cmp word [eax], 0x71     ; VK_F2
0x00682109  jne  0x6821bc            ; 非F2跳走
0x0068210F  xor ebx, ebx
0x00682111  mov eax, 0x1869f         ; Random(99999+1) 上界
0x00682116  call 0x402f58            ; Random
0x0068211B  mov ebx, eax             ; EBX = challenge
0x0068211D  cmp ebx, 0x2710          ; >10000 才接受
0x00682123  jle  0x682111
0x00682125  lea edx, [ebp-0xc]
0x00682128  mov eax, ebx
0x0068212A  call 0x4098f0            ; IntToStr(challenge)
0x0068212F  lea ecx, [ebp-0xc]
0x00682132  mov edx, 0x682494        ; "Please Input password"
0x00682137  mov eax, 0x6824b4        ; "N9056"
0x0068213C  call 0x43c0c8            ; InputBox(N9056, "Please Input password", challenge默认)
;  ... 检查输入非空, StrToInt 到 ESI ...
```

**flag 内存路径**：`[0x006B7990]` → 全局对象指针 → +0 偏移的 byte
（监视脚本 `_watch_eng_flag.py` 已固化）

```text
frac     = y - trunc(y)
password = trunc(frac * 100000)
```
