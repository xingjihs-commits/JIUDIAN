# CardLock F2 工程密码算法（已还原）

- **时间**: 2026-05-31
- **函数**: `0x00692754`（`edtHotelName` 按 F2 → `CMP ESI,EAX` @ `0x0068218F`）
- **KEY**: `0x176` = **374**（第二个参数 `EDX`）

## 结论（给用户）

| 对话框里的数（challenge） | 数字反转 | **应输入的密码** |
|--------------------------|----------|------------------|
| 66576 | 67566 | **65303** |
| 28756 | 65782 | **20809** |
| 18525 | 52581 | **11953** |

对话框**预填的是 challenge，不是密码**。直接点 OK 一定失败（静默返回）。

## 公式

```text
rev     = reverse_5_digits(challenge)   # 例: 66576 → 67566
x       = 374 + rev
password = Round( Frac( Sqrt(x) ) × 100000 )
```

`Frac` = 取小数部分（向零取整，Pascal `Frac`）。

## 通过后

校验成功会走 **DirectCardErase**（`DirectWriteUSB` 路径），与 E0035-3 回读校验无关。仍需按说明准备**总卡/工程授权卡**。
