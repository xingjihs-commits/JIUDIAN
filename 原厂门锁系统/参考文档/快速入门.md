## 五个关键数字

| 数字 | 含义 |
|:----:|:-----|
| `d12=1` | 轻量 USB 模式，能读卡发卡（我们用这个） |
| `d12=0` | 完整 USB 模式，返回 259（固件永久锁） |
| `259` | 发卡器 MCU 固件保护锁错误码 |
| `C92B20B7` | V9 卡 Magic 头（MIFARE 第 0-3 字节） |
| `0x68218F` | F2 补丁地址：`3B F0 → 3B C0`（工程模式永远通过） |
