**database.py 内部结构：**
```
ShadowDatabase 类 (1874行)
├── __init__: WAL + synchronous + cache + 表初始化
├── check_integrity(): PRAGMA integrity_check
├── checkpoint_wal(): WAL 写回
├── execute(): SQL执行 + 慢查询日志
├── transaction(): 显式事务
├── append_ledger(): 哈希链写入
├── verify_ledger_integrity(): 哈希链验证
├── get_config/set_config: 系统配置
├── _init_tables: 表创建 → TABLES → MIGRATIONS → _init_new_tables → run_schema_migrations
├── _table_exists(): 表存在性检查
├── _table_has_columns(): PRAGMA table_info
├── _migrate(): 补列迁移（跳过不存在表）
├── _init_new_tables(): 补建表 + 补列 + 索引
└── 委托方法: get_audit_overview→aggregation, run_night_audit→aggregation, 等
```

**依赖关系：**
```
database.py
  ├──→ db_schema.py (TABLES, TABLE_CREATE_ORDER, MIGRATIONS)
  ├──→ db_migration.py (run_schema_migrations)
  ├──→ secure_db.py (加密连接)
  ├──→ db_access/aggregation.py (聚合查询委托)
  └──→ services/risk_service.py (风控委托)

db_schema.py (627行) — 表结构定义
db_migration.py (550行) — 版本迁移引擎
secure_db.py (137行) — SQLCipher 加密连接
```

**修改指引：**
- 加新表 → 改 `db_schema.py:TABLES` 字典
- 加新列 → 改 `db_schema.py:TABLES` 对应表（建表时直接包含），同时在 `_migrate()` 加补列项
- 改迁移逻辑 → `db_migration.py`
- 加聚合查询 → `db_access/aggregation.py`
- 改哈希链 → `database.py:append_ledger()` + `core/ledger.py:LedgerHashChain`
- **改数据库后必须跑测试**：`pytest tests/test_database.py`

## 三、关键数据流向（5 条主线）

```
1. 入住流程:
   tabs/frontdesk/checkin_tab.py
   → core/guests.py:checkin() [trace:checkin]
   → database.py:transaction()
   → INSERT guests + UPDATE rooms
   → event_bus.guest_checkin.emit()

2. 退房流程:
   tabs/frontdesk/checkout.py
   → services/checkout_service.py:execute() [trace:checkout]
   → core/guests.py:checkout()
   → core/ledger.py:append()
   → core/inventory.py:deduct()
   → database.py:transaction()
   → event_bus.guest_checkout.emit()

3. 发卡流程:
   tabs/card_system/card_issue.py:_do_issue() [trace:card_issue]
   → services/card_service.py:issue_card()
   → lock_adapters/cardlock_auto.py
   → lock_adapters/bridge_client.py
   → lock_adapters/rfl_bridge_32.py (32位子进程)
   → V9RFL.dll

4. 夜审流程:
   tabs/night_audit_tab.py
   → audit_engine.py
   → db_access/aggregation.py:run_night_audit()
   → database.py:append_ledger(NIGHT_AUDIT)

5. 云端心跳:
   heartbeat_service.py (每5分钟)
   → cloud-worker/worker.js (POST /api/hotel-poll)
   → kill_switch / hotel_status 检查
```
